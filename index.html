hexo部署

搭建hexo博客，部署到github上（linux）

前期准备

在本地通过hexo构建网页

1.下载nodejs npm

hexo 对 nodejs版本有要求，越新的版本要求nodejs版本越高

访问nodejs官网下载Linux Binaries(x64)并放在一个任意的目录中，例如/opt

解压

配置环境变量

2.下载hexo

3.初始化hexo

在任意位置创建一个文件夹在文件夹中打开终端

完成初始化

生成一个默认的本地网页

如何通过SSH使用 git部署hexo到个人github上

git config --global user.name "xiongshialltheway"
git config --global user.email "lichaoyuecc@gmail.com"

生成属于本机的ssh公钥用来连接本地和你的github账号。

在本地的config文件中配置deploy信息。

在blog目录下安装插件hexo-deployer-git

即可部署完成，会有一些延迟。
